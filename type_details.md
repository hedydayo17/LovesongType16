## 1. バイブス警察(バイブス型)
- **キャッチ**:理屈より先に、心が動いてる。
- **カラー / モチーフ**:#FF5E7E / 稲妻・ネオン
- **相性**:ときめきパパラッチ・情熱ラブゾンビ
- **本文**:理屈より先に、心が動く。「この人いいかも」と感じた瞬間、もう恋は始まっている。第一印象とその場の空気を信じて飛び込めるのがあなたの強み。スピード感のある恋ほど燃えるし、ピンとこない相手には時間をかけない潔さもある。ノリで動いた結果、あとから「ちょっと早まったかな」と思う瞬間もあるけれど、その瞬発力こそが人を惹きつける魅力。直感に素直なあなたには、衝動を肯定してくれるアップテンポでエモい曲がよく似合う。同じ熱量で走れるタイプと相性がいい。

## 2. 運命マジシャン(運命型)
- **キャッチ**:この出会い、ぜったい意味がある。
- **カラー / モチーフ**:#7C5CFF / 星座・月
- **相性**:進撃のロマンチスト・ピュアエンジェル
- **本文**:「この出会いには意味がある」と感じられる人。偶然を必然に変える物語の感度が高く、タイミングや巡り合わせを大切にする。運命を信じているぶん、ここぞという相手には深く心を寄せられるし、待つことも怖くない。一方で「運命だから」と相手を理想化しすぎて、現実とのギャップに気づくのが遅れることも。でもその一途な信じる力は、関係を特別なものに育てる原動力になる。物語性のある曲、出会いを彩るドラマティックなメロディがあなたの恋にはよく馴染む。

## 3. 進撃のロマンチスト(ロマンティック型)
- **キャッチ**:どうせなら、映画みたいな恋がいい。
- **カラー / モチーフ**:#FF7AB6 / 薔薇・フィルム
- **相性**:運命マジシャン・ときめきパパラッチ
- **本文**:恋を一本の映画みたいに描きたい人。シチュエーションや雰囲気づくりにこだわり、何気ない日常も特別なシーンに変える演出力がある。理想の世界観がはっきりしているぶん、ときに現実とのギャップに戸惑うことも、夢を見すぎてしまうこともある。でもその「素敵なものを信じる力」が、関係に彩りと余韻を与える。あなたの恋には、ストーリーが浮かぶようなドラマティックな曲や、情景の見えるバラードがよく似合う。世界観を一緒に味わえる相手と特に響き合う。

## 4. 一途ペンギン(一途型)
- **キャッチ**:決めたら、わき目もふらない。
- **カラー / モチーフ**:#3AA0FF / ペンギン・氷
- **相性**:慎重うさぎ・ピュアエンジェル
- **本文**:一度好きになったら、まっすぐにその人だけを想える人。心が決まればぶれず、他に目移りもしない誠実さがあなたの軸。安定した関係を育てるのが得意で、相手に安心感を与えられる。一方で、想いが強いぶん「ずっとこのままがいい」と変化を恐れたり、尽くしすぎて自分を後回しにしてしまうことも。それでも、その変わらない愛情は何より信頼される強み。あなたの恋には、長く寄り添うような温かいミディアム曲や、まっすぐな愛を歌うナンバーがよく馴染む。

## 5. ヤキモチモンスター(ヤキモチ型)
- **キャッチ**:あなたを、誰にも渡したくない。
- **カラー / モチーフ**:#FF4D4D / 炎・ハート
- **相性**:一途ペンギン・情熱ラブゾンビ
- **本文**:「私だけを見てほしい」——その独占したい気持ちは、それだけ本気で相手を想っている証拠。好きな人への関心が人一倍強く、ちょっとした態度の変化にも敏感に気づける愛情深さがある。モヤモヤを抱え込みやすかったり、相手の人間関係が気になりすぎてしまう瞬間もあるけれど、その情熱はちゃんと伝えれば最大の魅力になる。あなたの恋には、感情をストレートにぶつけるエモーショナルな曲や、切なさを歌うナンバーが寄り添ってくれる。同じ熱量で応えてくれる相手と相性がいい。

## 6. 推し活ベビー(推し活型)
- **キャッチ**:あなたの幸せが、わたしの幸せ。
- **カラー / モチーフ**:#FF8FB1 / リボン・ペンライト
- **相性**:一途ペンギン・ピュアエンジェル
- **本文**:好きな人を応援し、喜ぶ顔のために動けるのがあなた。相手の好きなものを覚えて、さりげなく支える観察力とまめさは、誰にでもできることじゃない。尽くすこと自体が愛情表現で、与えることに幸せを感じられる。一方で、頑張りすぎて自分のことを後回しにしたり、見返りのなさに疲れてしまうことも。だからこそ、自分も大切にできると恋はもっと豊かになる。あなたの恋には、まっすぐ応援するような前向きな曲や、愛を惜しみなく歌うナンバーがよく似合う。

## 7. チル仙人(チル型)
- **キャッチ**:ほどよい距離が、いちばん心地いい。
- **カラー / モチーフ**:#36C9A8 / 山・雲
- **相性**:同志の虎・ミステリアス狼
- **本文**:ベタベタしすぎない、ちょうどいい距離感を大切にする人。自分の時間も相手の時間も尊重できて、余白のある関係を心地よく保てる。執着しないぶん、相手も自然体でいられる安心感がある。一方で、クールに見られて「ちゃんと好きなのか伝わってる?」と思われたり、踏み込むタイミングを逃すことも。でもその落ち着きは、長く続く関係をつくる土台になる。あなたの恋には、肩の力が抜けたチルな曲や、心地よく流れるグルーヴのある音楽がよく馴染む。

## 8. 慎重うさぎ(慎重型)
- **キャッチ**:急がない。でも、ちゃんと本気。
- **カラー / モチーフ**:#B79CFF / うさぎ・新月
- **相性**:一途ペンギン・同志の虎
- **本文**:恋にはじっくり時間をかけたい人。相手をよく見て、信頼できると確かめてから心を開く慎重さがあなたの誠実さ。勢いで動かないぶん、関係が始まれば長く大切に育てられる。一方で、考えすぎてタイミングを逃したり、なかなか前に進めずもどかしくなることも。でもその丁寧さは、相手に「ちゃんと向き合ってくれている」という安心を与える。あなたの恋には、静かに寄り添うアコースティックな曲や、じんわり染みるバラードがよく似合う。同じペースで歩める相手と心地よく進める。

## 9. 同志の虎(同志型)
- **キャッチ**:わかり合えるって、いちばん強い。
- **カラー / モチーフ**:#2EC4B6 / コンパス・天気図
- **相性**:チル仙人・マブダチエイリアン
- **本文**:ドキドキより「わかり合えること」を大切にする人。価値観や考え方が合う相手に強く惹かれ、語り合える関係に安心を感じる。感情に流されず、対等なパートナーシップを築けるのが強み。一方で、理屈で考えすぎて気持ちの盛り上がりを後回しにしたり、条件で人を見てしまう瞬間も。でもその誠実な向き合い方は、長く支え合える絆をつくる。あなたの恋には、歌詞の世界観が深い曲や、一緒に未来を語りたくなるような音楽がよく馴染む。同じ目線で歩める相手と響き合う。

## 10. マブダチエイリアン(友達の延長型)
- **キャッチ**:いちばんの友達が、いちばん好きな人に。
- **カラー / モチーフ**:#46C2FF / 宇宙人・二つの星
- **相性**:同志の虎・ピュアエンジェル
- **本文**:友達として仲良くなった人を、いつの間にか好きになっているタイプ。気取らない自然体の関係を築くのが得意で、一緒にいて楽な空気をつくれる。恋人になっても友達のような対等さが続くのが魅力。一方で、友達の延長すぎて踏み込めなかったり、関係を壊すのが怖くて告白を先延ばしにしてしまうことも。でもその飾らない距離感は、誰よりも深い信頼につながる。あなたの恋には、肩肘張らないポップな曲や、青春の匂いがするバンドサウンドがよく似合う。

## 11. ミステリアス狼(ミステリアス型)
- **キャッチ**:ぜんぶは、見せないでいたい。
- **カラー / モチーフ**:#6C63FF / 狼・霧
- **相性**:チル仙人・慎重うさぎ
- **本文**:本心をすぐには見せない、つかみどころのある人。少し謎めいているくらいがちょうどいいと感じていて、その余白が独特の魅力になる。簡単に心を開かないぶん、本当に信頼した相手にだけ見せる素顔のギャップが効く。一方で、距離を取りすぎて誤解されたり、自分の気持ちを言葉にするのが苦手で相手を不安にさせることも。でもその奥行きは、人を惹きつけてやまない。あなたの恋には、ムードのあるダークめな曲や、余韻の残る大人っぽいサウンドがよく似合う。

## 12. ド直球ザウルス(直球型)
- **キャッチ**:好きなら、言う。それだけ。
- **カラー / モチーフ**:#FF7A45 / 恐竜・矢印
- **相性**:情熱ラブゾンビ・ピュアエンジェル
- **本文**:好きになったら自分から動く、まっすぐな行動力の持ち主。駆け引きせず、気持ちをストレートに伝えられる潔さがあなたの最大の魅力。迷うより先に一歩踏み出せるから、チャンスを逃さない。一方で、勢いがありすぎて相手のペースを置き去りにしたり、ストレートさが空回りすることも。でもその裏表のなさは、何より信頼される。回りくどさのない、堂々としたあなたには、背中を押してくれるパワフルな曲や、まっすぐ気持ちを歌うアンセムがよく似合う。受け止めてくれる相手と相性がいい。

## 13. ときめきパパラッチ(ときめき重視型)
- **キャッチ**:心が震える瞬間を、追いかけてる。
- **カラー / モチーフ**:#FF5CA8 / カメラ・フラッシュ
- **相性**:バイブス警察・進撃のロマンチスト
- **本文**:恋にときめきと刺激を求める人。胸が高鳴る瞬間が大好きで、その一瞬のために動けるエネルギーがある。新鮮さを感じる関係ほど輝けるし、相手のドキドキさせる演出も得意。一方で、刺激が落ち着くと物足りなくなったり、安定した日常に退屈を感じてしまうことも。でも、そのときめきを楽しむ感性は、関係をいつも新鮮に保つ才能でもある。あなたの恋には、高揚感のあるダンサブルな曲や、胸が高鳴るキラキラしたポップスがよく似合う。同じテンションで楽しめる相手と弾む。

## 14. 情熱ラブゾンビ(情熱型)
- **キャッチ**:好きの気持ち、止まらない。
- **カラー / モチーフ**:#FF3D6E / ハート・炎
- **相性**:ド直球ザウルス・ヤキモチモンスター
- **本文**:愛情の温度が高く、好きの気持ちを言葉でまっすぐ伝えられる人。感情がそのまま行動になるから、相手には愛されている実感がちゃんと届く。一度燃えると一途に突き進む熱量があなたの魅力。一方で、気持ちが溢れて止まらなくなったり、温度差に傷ついてしまうことも。でもその全力で愛する姿勢は、関係に確かな手応えをくれる。あなたの恋には、感情を爆発させるようなエモーショナルな曲や、愛を高らかに歌うバラードがよく似合う。同じ熱で応えてくれる相手と最高に響き合う。

## 15. 沼っくま(沼り型)
- **キャッチ**:ハマったら、もう抜け出せない。
- **カラー / モチーフ**:#9B5DE5 / くま・沼
- **相性**:ミステリアス狼・情熱ラブゾンビ
- **本文**:一度ハマったら、とことん夢中になれる人。好きな人のことで頭がいっぱいになる没入感は、それだけ深く愛せるということ。相手の小さな一面まで愛おしく感じられる感受性があなたの魅力。一方で、のめり込みすぎて自分を見失ったり、相手に合わせすぎて疲れてしまうことも。でも、その全身で恋に浸れる力は、何にも代えがたい愛情の深さ。あなたの恋には、ループしたくなる中毒性のある曲や、切なさにどっぷり浸れるエモい音楽がよく似合う。深さを受け止めてくれる相手と溶け合える。

## 16. ピュアエンジェル(ピュア型)
- **キャッチ**:まっすぐな気持ちが、いちばんの武器。
- **カラー / モチーフ**:#FFAFCC / 天使・羽
- **相性**:一途ペンギン・マブダチエイリアン
- **本文**:気持ちに素直で、いつまでも初恋みたいな純度を持っている人。駆け引きをせず、まっすぐに人を好きになれる素直さがあなたの一番の魅力。相手を疑わず信じられる優しさが、安心できる関係をつくる。一方で、純粋すぎて傷つきやすかったり、理想と現実のギャップに戸惑うことも。でも、その澄んだ気持ちは何より人の心を動かす。あなたの恋には、まっすぐで温かいラブソングや、胸がきゅんとする爽やかなポップスがよく似合う。同じ素直さを持った相手と、穏やかに育っていける。
