## このイラストが果たす役割

このサイトは **「自分のタイプをスクショして SNS にシェアする」** ことが成功の中心動線。つまり**キャラクターの絵そのものが拡散される広告であり、ブランドの顔**になる。プロフィール画像・XヘッダーやLINEアイコンに使われる前提で設計する。

参考にする世界観:
- **ラブタイプ診断64**:明るくポップ、丸み、ペールトーン+ビビッド差し色、シール感
- **16personalities**:全身ポーズ、軽い物語性、わずかな表情差
- **ちいかわ / すみっコぐらし**:愛嬌のあるシンプル線、控えめな配色でも記憶に残る造形

避けたい方向:
- 萌え系・美少女キャラ(ターゲット外)
- 写実的・大人っぽすぎる(ブランドが「軽くて楽しい」ため)
- 線が細すぎる繊細系(縮小して使うのでつぶれる)

## 共通スペック(全16体に適用)

| 項目 | 仕様 |
|---|---|
| サイズ | 1024×1024px(正方形、透過PNG) ※可能ならSVGも同時納品 |
| 解像度 | 印刷不要、Web用RGB |
| 体長比 | 2.5〜3頭身(コミカル寄り) |
| 構図 | **全身正面立ち姿**(ポーズはタイプごとに微差。詳細は各タイプ仕様参照) |
| 余白 | キャラ周囲に左右上下10%の余白(クロップして使うため) |
| 線 | あり / 太め(2〜3pxで縮小しても残る太さ) |
| 塗り | フラット + 軽いハイライト1段(グラデは最小限) |
| 表情 | 笑顔ベース、目はぱっちり大きめ、口は小〜中(キャラごとに微差OK) |
| 影 | 落ち影は無し(キャラ単体で背景に馴染ませる) |

## アートスタイル指示

- **線画**:ペン質感(完全均一じゃない方が温かい)。ベタ塗り境界は #2A1830 系の濃い紫グレー(真っ黒は避ける)
- **彩色**:タイプごとの指定カラーを「メインの体色」として大胆に使う(タイプ識別がひと目で立つよう)
- **トッパー / アクセサリー**:タイプ性格を表すモチーフを頭/手/服のいずれかに1点だけ(盛りすぎない)
- **背景**:無し(透過)

## 納品物まとめ

### 必須
- 全16体 × 透過PNG(1024×1024、各キャラ1体ずつ、計16ファイル)
- ファイル名:`{連番}_{パロディ名英字}.png` 例:`01_vibes_keisatsu.png` / `15_numokkuma.png`

### 推奨(あればさらに嬉しい)
- 同16体のSVG版(Web用、ファイルサイズ軽量化のため)
- 各キャラの**胸から上トリミング版**(プロフィールアイコン用 / 512×512円形透過PNG)
- ラインスタンプ用のバリエーション表情(喜怒哀楽4種 × 推しキャラ数体 = 後日相談)

### ファイル提出形式
- ZIPで一括 or Googleドライブ共有
- レイヤー保持の元データ(AI/PSD/FIG)も納品時に同梱希望

## NGリスト(避けたい表現)

- 性的・グラマラスな身体表現(対象は10〜30代女性ターゲット)
- 萌え系・アニメ美少女調
- 既存キャラクター(ピカチュウ・ちいかわ等)の模倣に見えるもの
- 過度に細い線(縮小時に消える)
- 暗い背景必須の塗り(白背景でも自立すること)
- 文字情報(タイプ名・キャッチをイラスト内に直接入れない。タイポはWeb側で乗せる)

## 進行の流れ(提案)

1. **ラフ案**:16体のうち**代表3体**(沼っくま / バイブス警察 / ピュアエンジェル)で全身ラフを提出 → ファウンダー承認
2. **本制作**:OKならその3体を清書 → ファウンダー再承認
3. **量産**:残り13体を本制作
4. **微調整**:全16体納品後、Web実装時に出る不具合(縮小時の見え方等)を1回微修正

予算・納期はファウンダーと別途相談。
