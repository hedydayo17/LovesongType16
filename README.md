# 音楽診断プロジェクト

オンライン診断サイト「**ラブソング診断16**」。

ユーザーがブラウザ上で20問の質問に答えると、16タイプのうちの1つに分類され、その人にぴったりのラブソング(メイン10曲＋プレイリスト)がレコメンドされる。MBTI / 動物占い的体験のTikTok時代版。

## 仕様(2026-05-19時点)

| 項目 | 仕様 |
|---|---|
| 所要時間 | 約3分(質問数12〜15) |
| タイプ数 | 16(独立アーキタイプ方式) |
| 結果楽曲 | **メイン10曲(プールからランダム抽出)** ＋プレイリスト |
| 入力ステップ | 診断前に **(0) 生年月日(YYYY/MM/DD)入力 [必須・カード非表示] + (1) ジャンル選択(18から1〜5) [Skip可] + (2) 好きなアーティスト3組入力 [Skip可]** |
| 楽曲データ | **Type × Genre マトリックス**で管理。各セルプール30〜50曲 |
| 再シャッフル | 結果画面に「別の10曲を見る」ボタンを設置 |
| 楽曲リンク | 外部音楽サービス想定 |
| 体験設計 | TikTokシェア前提・モバイル最優先・軽量 |

## コンセプト由来

- 診断結果として「楽曲」をレコメンドする形は、音楽メディア発の診断として親和性が高い
- 診断結果をシェアする文化との相性が良い(TikTok/X)
