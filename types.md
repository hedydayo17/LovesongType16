## 命名思想

### 全体ルール
[rules.md](./rules.md) のポジティブ命名原則を踏襲。プロフに貼って嬉しい/シェアしたい かどうかが判定基準。

### 採用された語彙系統
- **動物・キャラ系**:ペンギン / うさぎ / 狼 / ザウルス / くま / エンジェル / ベビー / ゾンビ / エイリアン / モンスター
- **職業・専門家系**:警察 / マジシャン / 予報士 / パパラッチ / 仙人
- **パロディ系**:進撃の◯(『進撃の巨人』パロ。1個だけ採用しレア感維持)

### タイプ別の議論記録(2026-05-20)

- **#5 ヤキモチ型**:当初「進撃のヤキモチモンスター」だったが、「進撃の◯」を1個だけにしたいので「ヤキモチモンスター」に短縮
- **#11 ミステリアス型**:候補「ミステリーキャット」が出たが、(1)「ミステリー」は名詞でミステリアス(形容詞)と微妙にズレ、(2)『キャッツ』の Macavity: The Mystery Cat と固有名詞被り、で却下。「ミステリアス狼」(漢字混じりの可愛さ)に決着
- **#12 直球型**:「ド直球ガエル」も検討したが、「ザウルス」のドスドス突進感のほうが直球型の本気度に合致
- **#14 情熱型**:当初「暴走情熱ゾンビ」 → 「暴走」と「情熱」の温度系意味重複 + ゾンビ単体のネガ含意で「情熱ラブゾンビ」に修正(「ラブ」が緩衝材になりゾンビがコメディ寄り)
- **#15 沼り型**:「沼+動物」テンプレを一度通った後、造語「沼っくま」(沼った+くま)に決着。TikTok層の造語スラング感を取りに行った

## 判定設計上の注意(質問設計時に必須)

「**混ざりやすいペア**」が3グループある。質問は意識的に切り分ける:

| 混ざりやすいペア | 切り分けの軸 |
|---|---|
| **一途 / ヤキモチ / 沼り** | 主語と方向(一途=自分が他を見ない / ヤキモチ=相手を取られたくない / 沼り=自分が夢中になる) |
| **推し活 / 情熱** | 行動(尽くす) vs 感情温度(燃える) |
| **ロマンティック / バイブス / 運命** | 入り口は全部「直感・理想系」(ロマンティック=演出 / バイブス=その場のノリ / 運命=必然感) |

**判定方式**:各タイプにスコアを割り振り、合計点が最高のタイプを返す(軸ベース判定ではない)。
